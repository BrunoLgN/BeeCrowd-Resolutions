# BeeCrowd-Resolutions

O objetivo deste repositório é resolver todas os desafios referentes a Estruturas de dados do site BeeCrowd.

Como o BeeCrowd funciona ?
Desafios no modelo leetcode, recebemos uma entrada e temos que produzir uma saida valida de acordo com o que o enunciado pede, simples assim.
A inumeros desafios no site e focaremos exclusivamente em estruturas de dados, atualmente(1/11/25) contamos com 180 exercicios disponiveis.
